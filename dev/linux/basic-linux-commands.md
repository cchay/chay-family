# Linux Commands
* file/folders
  * mv - Move
  * mkdir - make directory
  * rmdir - remove direcotry
  * rm - remove a file
  * locate - find a file
  * cp - copy
* disk
  * df - display disk info
  * du - disk usage
* System 
  * uname -srv   - linux distro
  * lsb_release -a  - linux distro
  * cat /proc/version
  * dpkg -l ubuntu-server
  * dpkg -l ubuntu-desktop
  * finger - who is on the sytem
